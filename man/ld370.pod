=head1 NAME

ld370 - host-native MVS linker (IEWL replacement) + host-to-MVS transport

=head1 SYNOPSIS

B<ld370> [B<-v>] B<-o> I<out.bin> [B<-L> I<dir> B<-l> I<name>] [B<--include> I<name>]
[B<--entry> I<name>] [B<--unload> I<u>] [B<--xmit> I<x> [B<--dsn> I<ds>]]
[B<--name> I<n>] I<obj>...

B<ld370> B<--unload-from> I<member.lm> [B<--name> I<n>] [B<--unload> I<u>] [B<--xmit> I<x>]

B<ld370> B<--pack> I<n1>=I<m1.lm> [B<--pack> I<n2>=I<m2.lm> ...] [B<--unload> I<u>] [B<--xmit> I<x>]

=head1 DESCRIPTION

B<ld370> links one or more OS/360 object decks into an MVS load module on the
host, replacing the mainframe linkage editor (B<IEWL>); the member it produces is
byte-identical to IEWL's. It performs automatic library call against B<.a>
archives (see L<ar370(1)>), and can wrap the result for transport to MVS: an
IEBCOPY unloaded-PDS image (B<--unload>) and a TSO TRANSMIT / NETDATA file
(B<--xmit>, RECFM=FB80) that uploads cleanly and installs with C<RECV370>.

It is invoked automatically by L<cc370(1)> for the F<.lm>/F<.xmit> stages.

=head1 OPTIONS

=over 4

=item B<-o> I<out>

Output file. The extension selects the form: F<.o> a single object, F<.lm> a load
module image, F<.xmit> a packaged NETDATA transmit file.

=item B<-e>, B<--entry> I<name>

Set the module entry point.

=item B<-l> I<name>, B<-L> I<dir>

Auto-call the archive C<lib>I<name>C<.a>, searching directory I<dir>. Unresolved
strong external references fail the link unless B<--allow-unresolved> is given.

=item B<--include> I<name>

Force-include an archive member even if nothing references it.

=item B<--unload> I<u>

Also write the IEBCOPY unloaded-PDS image to I<u>.

=item B<--xmit> I<x> [B<--dsn> I<ds>]

Write a TSO TRANSMIT (NETDATA) file to I<x>, optionally naming the dataset I<ds>.

=item B<--name> I<n>

Set the PDS member name.

=item B<--unload-from> I<member.lm>

Wrap a pre-built load-module member (rather than linking objects).

=item B<--pack> I<name>=I<member.lm>

Bundle several load modules into one unload/xmit (repeatable).

=item B<-v>

Print the linker version.

=back
